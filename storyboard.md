# Storyboard til Procesvideo - P5 Projekt

## Tekniske specifikationer

### Green Screen Setup
- **Baggrund:** Ensfarvet grøn/blå screen
- **Lys:** 3-punkt lighting (key, fill, back) for at undgå skygger
- **Afstand:** Min. 1 meter fra skærm til person
- **Kamera:** Stable mount, 1080p minimum, 24-30fps

### Post-produktion
- **Software:** Premiere Pro / DaVinci Resolve / Final Cut Pro
- **Chroma key:** Nøglefunktion for green screen removal
- **Baggrunde:** 
  - Website screenshots
  - Figma files
  - Research-dokumenter
  - Animerede grafiske elementer
- **Transitions:** Clean cuts, fade-ins (undgå distraherende effekter)
- **Music:** Neutral background music (royalty-free)
- **Text:** Clean sans-serif font, god kontrast

### B-roll materialer
- Screen recordings af:
  - Trello board
  - Figma design process
  - Think Aloud tests (med blur)
  - Kode i VS Code
  - Website før/efter
- Billeder fra virksomhedsbesøg
- Screenshots af research

### Varighed
**Total:** 6:00-6:15 minutter (juster efter krav: 5-10 min)

## Shot List til filmdag

### Forberedelse
1. Test green screen setup
2. Test lydsætning (lav-mikrofon anbefales)
3. Forbered alle on-screen visuals
4. Print dialog/bullet points

### Optagelser (prioriteret rækkefølge)
1. **Intro/outro** (kritisk - optag flere takes)
2. **Hoveddialog-scener** (alle PKP-overgange)
3. **B-roll** (screen recordings kan optages separat)
4. **Backup-takes** (ekstra footage til fleksibilitet)

---

## Tips til eksekvering

### Før optagelse
- **Script:** Skriv fuldt script, men tillad naturlig dialog
- **Timing:** Øv timing med storyboard
- **Test:** Lav test-optagelse af green screen

### Under optagelse
- **Multiple takes:** Optag hver scene 2-3 gange
- **Variation:** Prøv forskellige vinkler/positioner
- **Safety:** Optag ekstra general B-roll

### Efter optagelse
- **Backup:** Gem alle raw files flere steder
- **Review:** Gennemse footage samme dag
- **Edit plan:** Lav redigeringsplan baseret på faktisk footage
